# 🌤️ 대한민국 날씨 대시보드 ('웨더맵 코리아') 통합 기획서 및 구현 계획

지금까지 논의된 서비스 기획 방향성과 프론트엔드 기술 스택(Tailwind CSS 포함)을 종합한 문서입니다.

---

## 1. 서비스 개요 및 핵심 차별점

기존 날씨 앱과 웹사이트들이 텍스트 기호와 숫자로 이루어진 정보를 제공하는 것에서 벗어나, **대한민국 전역의 기상 상황을 직관적인 시각 데이터로 '한눈에' 파악**할 수 있게 돕는 프리미엄 대시보드 형태의 날씨 플랫폼입니다.

- **핵심 가치**: "읽는 날씨"에서 "보는 날씨"로의 패러다임 전환
- **타겟 유저**: 전국 날씨를 빠르게 확인하고 싶은 직장인, 미세먼지 수치에 민감한 사용자, 사진 촬영 등 야외 활동을 즐기는 사람.

---

## 2. 주요 기능 (Key Features)

### 🗺️ A. 전국 반응형 기상 지도 (Interactive Weather Map)
- **데이터 기반 컬러링**: 대한민국의 행정구역(시/도 단위)을 인터랙티브 지도로 구현하고, 온도/강수량/미세먼지 수치에 따라 지역별 색상을 다르게 렌더링(히트맵 형태)합니다.
- **레이어 토글 기능**: 다음과 같은 정보 레이어를 제공하며 손쉽게 전환할 수 있습니다.
  - 🌡️ **기온 레이어**: 파란색 ~ 빨간색 히트맵
  - ☔ **강수량 레이어**: 비/눈 아이콘 및 강수량 척도 알림
  - 😷 **대기 질 레이어**: 미세먼지 단계별 색상 표시
  - ⚠️ **기상 특보 레이어**: 태풍, 폭우 등 주요 기상 주의보 지역 하이라이트

### ⏱️ B. 직관적인 타임 슬라이더 (Time Slider)
- 마우스 드래그를 통해 과거부터 미래 48시간의 기상 변화를 **애니메이션**으로 부드럽게 확인합니다.

### 📍 C. 위치 기반 초단기 브리핑 (Hyper-local Summary)
- 현재 접속하신 지역을 기반으로 텍스트 기반 날씨 한 줄 요약을 UI 상단이나 팝업으로 제공합니다.

---

## 3. UI/UX 디자인 방향성 (Design Aesthetics)

차별화된 **Premium Design** 경험을 제공하여 모던하고 감각적인 대시보드 뷰를 구성합니다.

- **다이내믹 백그라운드 (Dynamic Background)**: 시간대와 지역별 날씨 상태에 맞춰 은은한 배경 애니메이션 제공 (비, 눈, 맑음 모션 등).
- **글래스모피즘 (Glassmorphism)**: 방해받지 않는 지도 가시성을 위해 조작 패널은 반투명한 블러(Blur) 처리의 유리 질감으로 디자인.
- **마이크로 인터랙션**: 지역 클릭 및 요소 hover 시 세련된 이펙트 적용.

---

## 4. 프론트엔드 기술 스택 및 구조 (Tech Stack)

사용자 요청에 따라 **Tailwind CSS**를 메인으로 채택한 아키텍처입니다.

| 기능 구분 | 채택 및 추천 기술 | 목적 |
| :--- | :--- | :--- |
| **프레임워크** | `React` + `Vite` | 빠르고 가벼운 초기 로딩 및 컴포넌트 기반 UI 개발의 효율성 |
| **스타일링** | `Tailwind CSS` | **(확정)** 유틸리티 클래스 기반 디자인, 글래스모피즘(backdrop-blur)의 빠른 적용 |
| **애니메이션** | `Framer Motion` | UI 전환 및 타임 슬라이더 조작 시 부드럽고 전문적인 애니메이션 트랜지션 제공 |
| **지도 시각화** | `react-simple-maps` | Tailwind와 연동이 쉬운 가볍고 깔끔한 형태의 국내 행정구역 2D SVG 구현 |
| **날씨 데이터** | 기상청 API & AirKorea | 대한민국 영역에 맞춘 가장 정밀하고 확실한 단기예보 및 미세먼지 데이터 |
| **상태 관리** | `Zustand` | 복잡하지 않은 전역 상태(선택된 레이어, 타임라인 시간 변수 등)를 빠르고 가볍게 처리 |

### 폴더 및 핵심 파일 컴포넌트 설계 시안

- **`src/App.jsx`**: 메인 레이아웃 (전체 화면 지도 배치, 패널 오버레이 컨테이너). 기능 확장을 위한 최상단 컴포넌트.
- **`src/components/WeatherMap.jsx`**: `react-simple-maps`와 대한민국 GeoJSON 지리 데이터를 결합하여 히트맵 형식의 지도를 그리는 역할.
- **`src/components/ControlPanel.jsx`**: 기온/강수량/대기질 등을 변경할 수 있는 글래스모피즘 형태의 UI.
- **`src/services/weatherApi.js`**: 공공데이터포털(기상청) API와 통신하는 Fetch 로직 분리.

## 5. 🚀 시작 전 논의가 필요한 오픈 퀘스천 (Next Steps)

문서 내용에 맞춰 즉시 실제 코드를 생성(스캐폴딩)하기 위해 아래 사항들을 최종 확정해주시면 감사하겠습니다.

1. **프레임워크 동의**: 제안해 드린 `Vite + React` 조합으로 앱 생성을 시작해도 될까요? 
2. **지도 형태 안내**: Tailwind CSS와 잘 어울리고 커스텀이 쉬운 `react-simple-maps(2D)`로 개발을 먼저 시작하겠습니다.
3. **API 연동 문제**: 보유 중이신 **기상청 API 키(공공데이터포털)**가 있으신가요? 아직 없으시다면 초기 디자인 및 지도 상호작용은 '가짜 데이터(Mock Data)'를 활용해 구성하겠습니다.

의견 주시면 바로 프로젝트 파일들을 생성하고 화면 구성을 도와드리겠습니다.
